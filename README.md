# ヒューマニティ・オブ・ザ・デッド
魂だけは腐らせない2D横スクロールアクション。
3本のHPゲージを管理しながら敵ゾンビを倒してステージ最奥を目指すゲームです。

## 開発環境/制作体制
* **制作人数**:11人
* **エンジン**:Unity(C#)
* **制作時間**:2024/10/17~2025/1/10
* **役職**:プログラマ

## 担当箇所
* Player移動
* シーン遷移時に暗いままのバグ修正
* 軽量化
* 爆弾設置アクション
* BGM/SE選定

## 操作方法
Joy-Con(2本)の横持ちに対応しています。
* **移動**:スティック
* **爆弾設置**:Y/左
