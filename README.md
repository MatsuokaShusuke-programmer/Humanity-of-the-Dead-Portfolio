# ヒューマニティ・オブ・ザ・デッド
魂だけは腐らせない2D横スクロールアクション。
3本のHPゲージを管理しながら敵ゾンビを倒してステージ最奥を目指すゲームです。

## 開発環境/制作体制
* **制作人数**:11人
* **エンジン**:Unity(C#)
* **制作時間**:2024/10/17~2025/1/10
* **役職**:プログラマ

## 担当箇所
* 音
* UI

## 操作方法
詳しい遊び方は「チュートリアル」やオプションの「操作方法」をご確認ください。
* **左右移動**:A/D
* **ジャンプ**:W

* **上半身攻撃**:I
* **下半身攻撃**:K
* **イレイ**:J(ドロップ部位に近づいて)
* **イショク**:L(ドロップ部位に近づいて)

* **テキスト送り**:Space
* **テキストスキップ**:Tab
* **役に立ったり立たなかったりする攻略のヒント**:H
* **オプション(一時停止):G
